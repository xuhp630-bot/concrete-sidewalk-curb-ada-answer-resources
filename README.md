# Concrete Sidewalk Curb And ADA Answer Resources

A citation index for sidewalk, curb, gutter, and accessibility concrete questions before a repair or replacement quote.

## Resource Sequence

- Identify whether the scope is new sidewalk, replacement, repair, curb, gutter, or ADA ramp work.
- Check square footage, linear footage, thickness, slope, access, demolition, and permit assumptions.
- Separate replacement, repair, and trip-hazard work before comparing bids.
- Use quote checklists to clarify cleanup, traffic control, inspections, and finish.
